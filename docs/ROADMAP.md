# MyLang 路线图

## 已完成

### 语言核心
- [x] 基本类型（int, bool, string, char, unit）
- [x] 复合类型（list, tuple, array, record）
- [x] ADT/GADT
- [x] 模式匹配
- [x] 一等函数、闭包、递归
- [x] 引用类型（ref, !, :=）
- [x] 异常处理（try/raise）

### 类型系统
- [x] Hind-Milner 类型推断
- [x] 泛型多态
- [x] Traits（类型类）

### 执行后端
- [x] 解释器
- [x] 字节码 VM（栈式）
- [x] 寄存器 VM
- [x] JIT x86-64
- [x] WASM 后端

### 运行时
- [x] 分代垃圾回收
- [x] Actor 并发模型
- [x] 代数效果

### 工具链
- [x] REPL
- [x] LSP 语言服务器
- [x] 包管理器
- [x] 增量编译
- [x] 调试器

### 标准库
- [x] 200+ 内置函数
- [x] JSON 支持
- [x] 网络/加密/并发
